# Pixel — Hiring landing page

Single-page marketing site for **Pixel**, built with **React**, **Vite**, and **Tailwind CSS**.

## Scripts

```bash
npm install    # dependencies
npm run dev    # local dev server (http://localhost:5173)
npm run build  # production bundle to dist/
npm run preview # serve production build locally
```

## Structure

- `src/components/` — layout sections (`Navbar`, `Hero`, `Features`, …)
- `src/components/ui/` — reusable `Button`, `Pill`, `SectionHeading`
- `src/data/` — copy and lists (features, process steps, testimonials, footer)

Edit content in `src/data/*.js` without touching JSX.

## Stack

- [Vite](https://vite.dev/) + React 18 (JavaScript)
- [Tailwind CSS](https://tailwindcss.com/) v3
- [lucide-react](https://lucide.dev/) icons
